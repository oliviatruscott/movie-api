API Endpoints

Buisness Logic URL HTTP Method Query Parameters Request body data format Response body data format
Get a list of all movies /movies GET - - A JSON object holding data about all the movies
Get a specific movie by title /movies/:title GET :title - A JSON object holding data about a single movie
Get genre by name /movies/genre/:genreName GET :Name - A JSON object holding data about the genre of a movie
Get director data by name /movies/director/:directorName GET :name - A JSON object holding data of a director
Allow new users to register /users POST - A JSON object holding data about the users to add, structured like this { "name" : "deven", "Password": "1234", "Email": ""deven27@gmail.com", "Birthday": "05-06-1999" } A JSON object holding data about the user that was added, structured like this { "name" : "deven", "Password": "1234", "Email": "devon27@gmail.com", "Birthday": "05-06-1999" } including an ID
Allow new users update their user info /users/:id PUT :id - A JSON object holding data about the updated information
Add movie to favorite list /users/:id/:movieTitle POST :id, :movieTitle - A JSON Object holding data about the User and the movie that was added
Delete movie from favorite list /users/:id/:movieTitle DELETE :id, :movieTitle - A JSON Object holding data about the User and the movie that was removed
Allow existing users to deregister /users/:id DELETE :id - A text message indicating whether the User has deregistered
